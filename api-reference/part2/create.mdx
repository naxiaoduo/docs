在本节中，我们将介绍如何优化和使用图片以及插入外部链接和内部链接。

### 2.2.1 图片的调整、对齐和缩放

Markdown 可以使用感叹号（!）和方括号（[]）来插入图片。这部分，我们前面已经介绍过，例如：

```
![图片替代文本](https://example.com/image.png)

```

您可以在插入图片后，对其进行调整、对齐。例如：

```xml
![图片描述](图片链接){.class名}

<!-- 示例 -->
![美女照片](https://example.com/image.jpg){.center}

```

其中， ` .class名 `  可以是下列之一：

 -  ` .left ` ：使图片左对齐

 -  ` .center ` ：使图片居中对齐

 -  ` .right ` ：使图片右对齐

在 Markdown 中，您可以使用 HTML 标签的  ` <img> `  来插入图片，并通过设置  ` width `  和  ` height `  属性来指定图片的尺寸。例如：

```html
<img src="图片链接" alt="图片描述" width="宽度" height="高度" />

<!-- 示例 -->
<img src="https://example.com/image.jpg" alt="美女照片" width="500" height="300" />

```

您可以使用相对或绝对的像素值或百分比来调整图片的大小。例如：如果您想将图像的宽度设置为 50%：

```html
<img src="图片链接" alt="替代文本" title="额外信息" width="50%" />

```

此外，您还可以使用 CSS 样式属性来对图片进行缩放。通过设置  ` style `  属性并使用  ` transform `  属性，您可以缩放图片的大小。例如：

```html
<img src="图片链接" alt="图片描述" style="transform: scale(缩放比例);" />

<!-- 示例 -->
<img src="https://example.com/image.jpg" alt="美女照片" style="transform: scale(0.8);" />

```

在上述示例中，将  ` 缩放比例 `  替换为所需的缩放比例，例如  ` 0.8 `  表示将图片缩小为原始尺寸的 80%。

以下是相关属性的说明：

 -  ` <img> `  标签用于定义图片，它是一个空标签，只包含属性，并且没有闭合标签。

 -  ` src `  属性指定图片的 URL 地址。

 -  ` alt `  属性用于为图像定义一串可替换的文本，当无法加载图像时，替代文本将告诉读者丢失的信息。

 -  ` title `  属性描述了元素的额外信息，通常作为工具提示使用。

 -  ` width `  和  ` height `  属性用于设置图像的宽度和高度。

请注意，在使用 HTML 标签时，确保在 Markdown 解析器中启用了 HTML 标签的解析功能。

### 2.2.2 插入外部链接和内部链接

Markdown 允许您插入外部链接和内部链接，使读者可以点击链接并访问相关网页或文档。它使用方括号（[]）和圆括号（）来插入外部链接和内部链接。例如：

外部链接示例：

```
[链接文本](链接URL)

<!-- 示例 -->
[了解更多信息](https://example.com)

```

内部链接示例：

```
[链接文本](文件路径)

<!-- 示例 -->
[查看第一部分](part1.md)

```

此外，你也可以使用 HTML 语法来插入外部链接和内部链接。例如：

```html
<a href="https://example.com">链接文本</a>
<a href="/path/to/internal/page.html">链接文本</a>

```

### 2.2.3 锚点的使用和跳转链接

锚点和跳转链接可以帮助读者快速定位文章中的不同部分，也可以帮助搜索引擎更好地抓取文章的内容。

使用 Markdown 的锚点语法，您可以在文章中创建锚点，以便在同一页面内实现跳转链接。锚点是在文章中指定的特定位置。

创建锚点：

```html
<a name="anchor-name"></a>

```

跳转链接到锚点：

```
[跳转到锚点](#anchor-name)

```

例如，在文章中的某个位置创建一个锚点：

```html
<a name="section1"></a>

```

然后，您可以在文章中的其他位置创建一个链接，以便跳转到该锚点：

```
[跳转到第一部分](#section1)

```

Markdown 也可以使用 HTML 语法来创建锚点和跳转链接。例如：

```html
<a id="anchor-name"></a>

[跳转到锚点](#anchor-name)

```

总之，掌握了 Markdown 的图片和链接优化技巧，你可以轻松地插入和调整图片，插入外部链接和内部链接，创建锚点和跳转链接，使文章看起来更加美观大方，也更加便于阅读。
