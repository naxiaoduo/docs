地图嵌入是一种在Markdown文档中添加地图的方式，可以帮助读者更好地理解地理位置或路线。通过嵌入地图，你可以展示特定位置的地理信息，使读者能够直观地了解地点的位置和周围环境。在本节中，我们将介绍如何在Markdown中嵌入Google地图、百度地图和高德地图。

### 2.4.1 嵌入 Google 地图

要嵌入 Google 地图，您需要先获取地图的嵌入代码。

1. 打开 Google 地图网站，找到您要嵌入的地图。
2. 单击地图左上角的菜单按钮，然后选择“分享”或“嵌入地图”。
3. 在弹出的对话框中，选择“嵌入地图”选项卡，然后复制地图的嵌入代码。

在 Markdown 文档中，使用  ` <iframe> `  标签嵌入 Google 地图。

```html
<iframe src="https://maps.google.com/maps?q=116.46,39.92&amp;amp;z=15&amp;amp;output=embed">
</iframe>

```

 - 将 “[https://maps.google.com/maps?q=116.46,39.92&amp;amp;z=15&amp;amp;output=embed”](https://maps.google.com/maps?q=116.46,39.92&amp;amp;z=15&amp;amp;output=embed%E2%80%9D ) 替换为您要嵌入的地图的链接。

 - 将 “116.46,39.92” 替换为地图的中心点坐标。

 - 将 “15” 替换为地图的缩放级别。

### 2.4.2 嵌入百度地图

要嵌入百度地图，您需要先获取地图的嵌入代码。

1. 打开百度地图网站（[https://map.baidu.com/），找到您要嵌入的地图。](https://map.baidu.com/%EF%BC%89%EF%BC%8C%E6%89%BE%E5%88%B0%E6%82%A8%E8%A6%81%E5%B5%8C%E5%85%A5%E7%9A%84%E5%9C%B0%E5%9B%BE%E3%80%82 )
2. 单击地图左上角的菜单按钮，然后选择“分享”或“嵌入地图”。
3. 在弹出的对话框中，选择“嵌入地图”选项卡，然后复制地图的嵌入代码。

在 Markdown 文档中，使用  ` <iframe> `  标签嵌入百度地图。

```html
<iframe src="https://map.baidu.com/staticmap?center=116.46,39.92&amp;zoom=15&amp;size=600x400&amp;markers=116.46,39.92&amp;key=你的百度地图密钥">
</iframe>

```

 - 将 “116.46,39.92” 替换为地图的中心点坐标。

 - 将 “15” 替换为地图的缩放级别。

 - 将 “600x400” 替换为地图的宽度和高度。

 - 将 “116.46,39.92” 替换为地图上的标记。

 - 将 “你的百度地图密钥” 替换为您的百度地图密钥。

**百度地图 API 定制工具**

百度地图 API 定制工具（[http://api.map.baidu.com/lbsapi/creatmap/index.html）可以帮助您快速、轻松地创建自定义地图。您可以使用该工具添加自己的数据、标记和样式，以创建满足您特定需求的地图。](http://api.map.baidu.com/lbsapi/creatmap/index.html%EF%BC%89%E5%8F%AF%E4%BB%A5%E5%B8%AE%E5%8A%A9%E6%82%A8%E5%BF%AB%E9%80%9F%E3%80%81%E8%BD%BB%E6%9D%BE%E5%9C%B0%E5%88%9B%E5%BB%BA%E8%87%AA%E5%AE%9A%E4%B9%89%E5%9C%B0%E5%9B%BE%E3%80%82%E6%82%A8%E5%8F%AF%E4%BB%A5%E4%BD%BF%E7%94%A8%E8%AF%A5%E5%B7%A5%E5%85%B7%E6%B7%BB%E5%8A%A0%E8%87%AA%E5%B7%B1%E7%9A%84%E6%95%B0%E6%8D%AE%E3%80%81%E6%A0%87%E8%AE%B0%E5%92%8C%E6%A0%B7%E5%BC%8F%EF%BC%8C%E4%BB%A5%E5%88%9B%E5%BB%BA%E6%BB%A1%E8%B6%B3%E6%82%A8%E7%89%B9%E5%AE%9A%E9%9C%80%E6%B1%82%E7%9A%84%E5%9C%B0%E5%9B%BE%E3%80%82 )

**百度地图 JavaScript API GL**

百度地图 JavaScript API GL 是一款使用 WebGL 技术开发的地图 API。它可以帮助您创建具有 3D 效果的地图。百度地图 JavaScript API GL 提供了丰富的功能，包括地图展示、定位、覆盖物、检索、路线规划等。

要使用百度地图 JavaScript API GL，您需要先注册一个百度地图开发者账号。注册完成后，您就可以使用该 API 创建 3D 地图了。

**注意：**

在使用百度地图 API 之前，您需要先申请密钥（ak）。您可以通过以下步骤申请密钥：

1. 打开百度地图 API 申请页面：[http://lbsyun.baidu.com/apiconsole/key](http://lbsyun.baidu.com/apiconsole/key )
2. 登录您的百度账号。
3. 单击“创建应用”按钮。
4. 在弹出的对话框中，填写相关信息，然后单击“提交”按钮。
5. 等待审核通过后，您就可以获取密钥了。

### 2.4.3 嵌入高德地图

要嵌入高德地图，您可以使用高德地图的嵌入工具：[https://lbs.amap.com/tools/card。](https://lbs.amap.com/tools/card%E3%80%82 )

步骤如下：

1. 填写位置信息并定位标注点（加*的为必填内容）

 - 位置：输入地图的中心点坐标或地址。

 - 标注点：在地图上添加标注点。

2. 设置信息窗口

 - 状态：选择信息窗口的显示状态。

 - 图片：添加信息窗口中的图片。

 - 电话：添加信息窗口中的电话号码。

 - 简介：添加信息窗口中的简介。

3. 设置地图尺寸

 - 选择地图的大小。

 - 如果选择“自定义”，则可以手动输入地图的宽度和高度。

- 点击“提交”。

4. 生成代码

 - 复制生成的代码。
-  将 ` <iframe> ` 标签代码嵌入到 Markdown 中

以上是在 Markdown 中嵌入 Google 地图、百度地图和高德地图的方法。通过使用这些技巧，你可以以交互式和可视化的方式展示地理信息，使你的文档更加生动和具有吸引力。
