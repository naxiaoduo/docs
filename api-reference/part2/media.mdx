在Markdown中，您可以轻松地插入音频、视频和其他在线内容，以增强您的文档的可视性和吸引力。

### 2.3.1 插入音频和视频

要插入音频和视频文件，您可以使用 HTML 的  ` <audio> `  和  ` <video> `  标签，并使用 Markdown 的 HTML 语法将其嵌入到文档中。

例如，如果您在电脑中使用 Obsidian 本地软件写作文档，需要插入一首存在电脑中的歌曲，您可以使用以下语法：

```html
<audio src="音频文件路径" controls></audio>

```

```html
<audio src="/Users/username/Music/my_music.mp3" controls></audio>

```

在上述示例中，使用  ` <audio> `  标签可以将音频嵌入到文档中，并添加  ` controls `  属性以显示音频控件，例如播放按钮和音量控制。这将在文档中嵌入一个音频播放器，允许您播放歌曲。

同样的方式，您可以使用  ` <video> `  标签来插入视频文件：

```html
<video src="视频文件路径" width="800px" height="600px" controls="controls"></video>

```

```html
<video src="/Users/username/Videos/my_video.mp4" width="800px" height="600px" controls="controls"></video>

```

使用  ` <video> `  标签可以将视频嵌入到文档中，并添加  ` controls `  属性以显示视频控件，例如播放按钮和进度条。

您可以使用以下属性来控制音视频的显示和播放：

 -  ` src ` ：要嵌入页面的视频的 URL。

 -  ` controls ` ：允许用户控制音视频的播放，包括音量、跨帧、暂停/恢复播放。

 -  ` width ` ：视频显示区域的宽度，单位是 CSS 像素。

 -  ` height ` ：视频显示区域的高度，单位是 CSS 像素。

### 2.3.2 嵌入在线内容，如网易云音乐和哔哩哔哩视频

除了插入本地媒体文件之外，Markdown 还支持嵌入在线内容，如网易云音乐和哔哩哔哩视频。这可以通过使用 HTML 的  ` <iframe> `  标签来实现，该标签可以将另一个 HTML 页面嵌入到当前页面中。

#### 嵌入网易云音乐

要嵌入网易云音乐，可以使用以下 HTML 代码：

```html
<iframe src="https://music.163.com/outchain/player?type=2&amp;id=网易云音乐歌曲ID&amp;auto=1&amp;height=66" width="330" height="86" frameborder="no"></iframe>

```

其中，网易云音乐歌曲 ID 是你要嵌入的歌曲的 ID。将宽度和高度替换为所需的显示尺寸，通过使用  ` <iframe> `  标签并设置相应的属性，您可以在 Markdown 文档中嵌入网易云音乐。

示例：

以下是一个嵌入网易云音乐歌曲的 Markdown 示例：

```html
<iframe src="https://music.163.com/outchain/player?type=2&amp;id=2092573643&amp;auto=1&amp;height=66" width="330" height="86" frameborder="no"></iframe>

```

#### 嵌入哔哩哔哩视频

要嵌入哔哩哔哩视频，可以使用以下 HTML 代码：

```html
<iframe src="//player.bilibili.com/player.html?aid=视频ID" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="宽度" height="高度"></iframe>

```

其中，哔哩哔哩视频 ID 是你要嵌入的视频的 ID。

示例：

以下是一个嵌入哔哩哔哩视频的 Markdown 示例：

Copy

```html
<iframe src="//player.bilibili.com/player.html?aid=537416597" scrolling="no" border="0" frameborder="no" framespacing="0" allowfullscreen="true" width="宽度" height="高度"></iframe>

```

**注意：**

 - 在 Markdown 中嵌入在线内容时，需要确保所嵌入的内容是公开可访问的，否则可能会导致嵌入失败。

 - 具体的嵌入语法可能因为不同的嵌入内容而有所不同。在使用特定的嵌入内容（如网易云音乐或哔哩哔哩视频）时，请根据它们的嵌入指南或文档提供的具体语法进行操作。

 - 嵌入的在线内容也可能会因为版权或其他原因而被移除，因此在嵌入时需要注意风险。
