#### **故事**

自从2018年以来，我在语雀上写了许多笔记和知识文档。语雀是一款云笔记软件，用起来很方便，但我也担心内容安全问题。有一天，语雀突然出现故障，网站都打不开了。这让我意识到，把所有笔记都放在云端并不是一个好主意。

我开始寻找一款本地笔记软件，可以让我安全地保存笔记，并且能够满足我对内容写作的需求。经过一番对比，我选择了Obsidian。

Obsidian是一款使用Markdown语法的笔记软件，它可以实现双向链接，并可以将笔记之间的双向链接可视化为神经网络图。这使得我可以轻松地将笔记之间建立联系，并创建知识网络。

我开始使用Obsidian来管理我的笔记和文档。Markdown的语法很简单，而且Obsidian提供了一些非常棒的功能来帮助我组织和管理笔记。我逐渐爱上了Markdown，并使用[emlog](http://www.emlog.net/ )（支持Markdown编辑器的云端内容管理系统）建立一个个人网站，开始使用Markdown来撰写我的文章等。

Markdown的语法简单明了，而且它可以导出为多种格式，如HTML、PDF、Word等。这使得Markdown成为了一种非常通用的写作工具。

我希望本书能够帮助您像我一样爱上Markdown，并使用Markdown来创作出各种各样的精彩内容。

**使用Markdown的好处：**

- **简单易学：** Markdown 的语法非常简单易学，您可以在短时间内掌握它的使用方法。

- **轻量级：** Markdown 是一种轻量级的标记语言，它不会增加文档的大小，也不会影响文档的加载速度。

- **通用性强：** Markdown 可以导出为多种格式，如 HTML、PDF、Word 等，这使得它成为了一种非常通用的写作工具。

- **跨平台：** Markdown 可以跨平台使用，您可以在任何平台上使用 Markdown 编辑器来撰写文档。
